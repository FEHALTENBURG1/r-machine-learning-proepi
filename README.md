# Curso de Análise de Dados e Machine Learning com R para Saúde

> Imersão prática em Ciência de Dados aplicada à saúde pública com dados do DATASUS.

**ProEpi 2026** · Autora: Danielly Xavier · [Lattes](https://lattes.cnpq.br/2373300932860701)

---

## Sobre o Curso

O treinamento percorre o ciclo de análise de dados em saúde: preparação do ambiente, obtenção de dados, pré-processamento, análise exploratória, estatística inferencial e modelagem em aprendizado de máquina.

- **Carga horária:** 20 horas
- **Público:** estudantes e profissionais interessados em análise de dados e ML em saúde pública
